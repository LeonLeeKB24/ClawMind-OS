// core modules

The agent brain stack

🧠

Persistent Memory

Three-tier memory architecture (HOT/WARM/COLD) with on-demand injection. Agents carry context across sessions — no more starting from zero.

⚙️

Task State Machine

Agents track task progress across interruptions. Resume from any checkpoint without replaying the entire context window.

📊

Self-Evaluation

After each task, agents score their own performance, identify failure patterns, and update their strategy — autonomous improvement.

🔗

Multi-Agent Comms

Standardized interfaces for agent-to-agent message passing. Orchestrators, workers, and specialists can coordinate reliably.

// memory architecture

Token-efficient by design

HOT

Active Context

Always injected. Current task, recent decisions, active agent state. <500 tokens.

WARM

Recent Memory

Injected on relevance match. Last 7 days of episodic memory, learned preferences.

COLD

Long-term Archive

Retrieved by semantic search only. Unlimited storage, minimal token cost.
